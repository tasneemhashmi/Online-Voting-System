# Online-Voting-System
The Online Voting System is a console-based Java application designed to simulate a secure and efficient voting process. The system allows users (voters) to cast votes for candidates, ensures that each voter can vote only once, and calculates the final results by counting votes.

This project uses Object Oriented Programming fundamentals and concepts like encapsulation, class design, and method implementation to interact with the user (voter) and seek their voter ID and candidate choice to check whether the user is eligible to vote, has voted or not to prevent duplicate voting by maintaining a collection (ArrayList and/or HashSet) to store voter ID. When a voter inputs his voter ID, the collection is searched through to determine whether the voter has voted. If not, program runs further by allowing the voter to vote for his candidate of choice. The vote count for the chosen candidate is incremented and stored in a collection of candidates-to-vote count mapping. The final result is then displayed. The system uses file handling to store the voting data efficiently for further analysis and to ensure a permanent record is maintained, thereby supporting real-time data integration.
